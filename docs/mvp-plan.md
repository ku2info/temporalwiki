# MVP Plan

## PR 1: Foundation

- Documentation.
- Next.js project setup.
- Domain model types.
- Temporal resolution service.
- Unit tests.

## PR 2: Viewer

- Full-screen atlas viewer.
- Timeline and center entity switching.
- Group regions, cards, arrows, labels, and annotations.
- Empty and loading states.

## PR 3: Editing Basics

- Drag nodes.
- Save view node positions.
- Switch and save center entity and timeline point.
- Basic forms for timeline points, entities, relationships, groups, and annotations.

## PR 4: Persistence

- Prisma client integration.
- Import sample data into PostgreSQL.
- Repository interfaces and server actions.

## PR 5: Product Hardening

- Better responsive behavior.
- Keyboard operations.
- Error handling.
- Export/import format.
