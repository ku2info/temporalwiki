# Open Questions

- Should entity types remain fixed for the MVP, or should projects define custom entity types?
- Should relationship labels support center-perspective-specific phrasing in the domain model or only in view edges?
- Should editing be an overlay mode on the presentation canvas or a separate route?
- What import/export format should become canonical: JSON, YAML, CSV bundle, or database dump?
- How much automatic layout is needed before manual positioning becomes tedious?
- Should relationship state changes such as "enemy to ally" be modeled as one relationship with state changes or separate relationships with active ranges?
- What visual style range best fits the first real dataset?
- When authentication is added, what is the ownership model for projects and views?
