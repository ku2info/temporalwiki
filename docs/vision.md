# Vision

Temporal Atlas is a visual atlas for changing relationships. It helps people understand a knowledge world by shifting time, center perspective, and presentation layout.

The core experience is not reading an article. The core experience is looking at a relationship board, changing the center entity, moving through timeline labels, following edges, and reading short context where it helps the graph make sense.

The public surface should feel closer to an edited magazine spread, research board, scrapbook, or character relationship chart than a database admin screen. Editing tools can be more systematic, but the presentation view should keep the graph itself in the foreground.
